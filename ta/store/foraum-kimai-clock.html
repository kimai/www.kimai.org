KimaiClock is a macOS menu bar app that lets you track your time with Kimai directly from your desktop.

Features

  • тП▒я╕П Start & stop timesheets from the menu bar
  • ЁЯУК View and resume recent tasks
  • ЁЯФФ Idle detection reminders
  • ЁЯФФ App launch reminders (VS Code, PhpStorm, Xcode, тАж)
  • ЁЯУб Automatically sync timesheets with Kimai server
  • ЁЯУУ English & German localization
  • ЁЯФТ Secure, Apple-notarized build
  • тЪб Lightweight & fast
  • ЁЯУж No third party dependency
  • ЁЯМЩ Dark mode support
  • ЁЯУи Actively maintained

Download and install

Clone from GitHub or download the latest .dmg from the Releases page.

Homebrew (recommended)

KimaiClock can also be installed via our own Homebrew tap:

# add company tap
brew tap foraum-gmbh/foraum https://github.com/Foraum-GmbH/homebrew-foraum

# install cask from company tap
brew install --cask foraum-gmbh/foraum/kimai-clock

To remove KimaiClock and untap the repository:

# uninstall the app
brew uninstall --cask foraum-gmbh/foraum/kimai-clock

# remove the company tap
brew untap foraum-gmbh/foraum

Usage

Open the app, connect your Kimai server in Settings, and start tracking time right from the menu bar.

Need help? Create a GitHub issue or contact us via email: support@foraum.de
