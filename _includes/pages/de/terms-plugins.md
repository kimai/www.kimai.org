**2. Erlaubter Nutzungsumfang**

Mit einer Lizenz dürfen Sie das Plugin in genau einer Kimai-Installation („Endprodukt") für sich selbst oder für einen einzelnen Kunden verwenden.

**3. Was gilt als zulässiges Endprodukt?**

Ein zulässiges Endprodukt ist eine der folgenden Kimai-Installationen:

- Eine Kimai-Installation, die Sie selbst betreiben – für Ihr Unternehmen, Ihren Arbeitgeber oder sich selbst.  
  *Beispiel: Sie administrieren Kimai für Ihren Arbeitgeber oder für Ihr eigenes Unternehmen.*

- Eine Kimai-Installation, die Sie für einen einzelnen Kunden installieren oder administrieren, unabhängig davon, wo diese gehostet wird.  
  *Beispiel: Ihr Kunde beauftragt Sie mit der Installation, Verwaltung und Aktualisierung von Kimai für sein Unternehmen.*

Eine einzelne Lizenz darf zusätzlich in einer nicht-produktiven Umgebung (z. B. Test-, Staging- oder Entwicklungsumgebung) verwendet werden, sofern dort keine echte Zeiterfassung für den Produktivbetrieb stattfindet.

Nach Ablauf oder Kündigung des Abonnements gilt folgendes:

Dieses Modell wird als **Perpetual Fallback License** bezeichnet.

**6. Übertragung der Lizenz auf einen Kunden**

**Wichtig:** Nach der Übertragung dürfen Sie das Plugin nicht mehr selbst nutzen, es sei denn, Sie erwerben eine separate Lizenz für Ihre eigene Nutzung.

*Beispiel: Sie kaufen eine Lizenz für einen Kundeneinsatz und geben den Lizenzschlüssel an Ihren Kunden weiter. Wenn Sie das Plugin anschließend auch selbst nutzen möchten, benötigen Sie eine zweite Lizenz.*

**7. Verbot der Mehrfachnutzung**

Eine einzelne Lizenz darf nicht für mehrere Kunden oder mehrere produktive Kimai-Installationen gleichzeitig verwendet werden. Für jede zusätzliche Installation ist eine separate Lizenz erforderlich.

**8. Verbot der SaaS-Nutzung**

Die Nutzung des Plugins in einer Software-as-a-Service-Umgebung (SaaS) ist ohne erweiterte Lizenz nicht gestattet.

Als SaaS gilt jedes Modell, bei dem die Preisgestaltung von der Anzahl der Endnutzer abhängt – unabhängig davon, ob dies über Einzelplatzlizenzen (per seat) oder Volumenlizenzen erfolgt.

Vom SaaS-Verbot ausgenommen ist das Hosting von Kimai-Installationen für einzelne Kunden, sofern je Kundeninstallation eine eigene Lizenz vorliegt und die Abrechnung nicht nutzerzahlabhängig erfolgt.

*Beispiel: Ein Anbieter betreibt Kimai-Instanzen für fünf verschiedene Unternehmen. Jede Instanz benötigt eine eigene Lizenz. Wird hingegen eine einzelne Instanz mehreren zahlenden Nutzern angeboten (z. B. als Monatsabo pro Nutzer), gilt dies als SaaS und erfordert eine erweiterte Lizenz.*

Für SaaS-Nutzung kontaktieren Sie uns bitte vorab.

### Häufig gestellte Fragen

Die folgenden Fragen werden uns regelmäßig gestellt. Die Antworten ergänzen den Lizenztext und sollen die wichtigsten Punkte verständlich zusammenfassen.

**Kann ich das Plugin nach Ablauf meines Abonnements noch nutzen?**  
Ja. Sie dürfen die letzte Version, die vor Ablauf Ihres Abonnements veröffentlicht wurde, dauerhaft weiter nutzen. Für zukünftige Updates benötigen Sie ein aktives Abonnement.

**Darf ich das Plugin in einer Test- oder Entwicklungsumgebung einsetzen?**  
Ja. Eine Lizenz deckt zusätzlich eine nicht-produktive Umgebung ab (z. B. Staging, lokale Entwicklung), solange dort keine echte Zeiterfassung im Produktivbetrieb stattfindet.

**Ich installiere Kimai für meinen Kunden. Brauche ich eine eigene Lizenz oder kauft mein Kunde eine?**  
Sie können die Lizenz selbst erwerben und an Ihren Kunden übertragen (inkl. Lizenzschlüssel). Danach dürfen Sie das Plugin nicht mehr selbst nutzen. Wenn Sie das Plugin auch in Ihrem eigenen Betrieb einsetzen möchten, benötigen Sie eine zweite Lizenz.

**Ich betreue mehrere Kunden. Kann ich eine Lizenz für alle verwenden?**  
Nein. Jede produktive Kimai-Installation benötigt eine eigene Lizenz – unabhängig davon, ob Sie der Betreiber sind oder Ihr Kunde.

**Was ist der Unterschied zwischen Hosting für Kunden und SaaS?**  
Hosting für Kunden bedeutet: Sie betreiben eine Kimai-Instanz für einen einzelnen Kunden, der dafür eine Pauschale zahlt. Das ist mit einer eigenen Lizenz je Instanz erlaubt.
SaaS bedeutet: Die Preisgestaltung hängt von der Nutzeranzahl ab (z. B. pro Nutzer und Monat, Volumenpakete). Das erfordert eine erweiterte Lizenz – bitte kontaktieren Sie uns in diesem Fall.

**Kann ich eine Lizenz weiterverkaufen?**  
Ja, Sie dürfen eine Lizenz einmalig an einen Kunden weitergeben oder weiterverkaufen. Nach der Übertragung verlieren Sie das Nutzungsrecht. Eine Weiterverteilung des Plugins selbst (z. B. als Teil eines eigenen Produkts) ist nicht erlaubt.
