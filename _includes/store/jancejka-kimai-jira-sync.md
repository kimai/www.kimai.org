KimaiJiraSync is an integration plugin for Kimai 2 and Atlassian Jira.

It automatically synchronizes worklogs from Kimai to Jira whenever a timesheet entry is created, updated or deleted. It also supports importing open Jira tasks into Kimai as activities, helping teams keep project and reporting workflows aligned across both systems.

The plugin supports flexible Jira issue mapping via activity meta fields or description prefixes, per-user and per-project Jira credentials, optional time multipliers and encrypted Jira API token storage.

KimaiJiraSync can be used **free** of charge with **one Kimai project**, which makes it easy to evaluate in a real environment. A paid **subscription** is required for **multi-project** usage.
