Підтримує Ваш бізнес

У Kimai є усі основні бізнес-функції, які необхідні для налаштування робочих процесів з обліку часу та інтеграції з наявною інфраструктурою.

Аутентифікація та безпека

Kimai підтримує зовнішні постачальники ідентичності та вхід через LDAP та SAML на кількох платформах (таких як Google Workspace, Azure AD або Authentik). Додатково можна включити двофакторну аутентифікацію за допомогою TOTP-токенів.

Звіти та аналізи

Аналіз записаного часу за користувачами, клієнтами, проєктами, видами діяльності, мітками, періодами часу та іншим.

Виставлення рахунків

Гнучкі шаблони рахунків-фактур (інвойсів), опції групування записів, налаштування формату номерів інвойсів та різноманітні формати файлів разом з PDF та DOCX.

JSON API

Завдяки розширеному JSON API для читання та запису даних зовнішні програми та спеціальні інструменти можуть обмінюватися даними з Kimai.

Що Kimai може зробити для Вас?

Kimai пропонує набагато більше, ніж просто облік часу, тут список інших важливих функцій для бізнесу.

Плагіни

Kimai побудований з можливістю розширення функціональності — за допомогою плагінів-розширень. Розробники, які хочуть їх створити, можуть використовувати багато точок розширення.

Кілька користувачів

Kimai може використовуватися будь-якою кількістю користувачів: кожна інсталяція підтримує необмежену кількість користувачів з різними ролями (див. дозволи).

Мови

Kimai може бути локалізований на будь-яку мову і вже перекладений більш ніж на 30 мов.

Команди

Команди об'єднують користувачів у логічні групи на кшталт департаментів. Обмежте доступ до даних, призначивши клієнтів і проєкти своїм командам.

Клієнти

Керуйте своїми клієнтами та додавайте адресу, контакти, країну, валюту, видимість (для архівування) та багато інших полів.

Проєкти

Керуйте проєктами для кожного клієнта з номером замовлення, бюджетом та видимістю (для архівування).

Виставлення рахунків

Гнучкі шаблони рахунків-фактур (інвойсів), опції групування записів, налаштування формату номерів інвойсів та різноманітні формати файлів разом з PDF та DOCX.

Експортувати дані

Дані табеля можна експортувати та надсилати у форматах XLSX, PDF, CSV та HTML. Експортовані записи можна заблокувати та захистити від змін.

Видатки

Придбайте розширення для відстеження витрат за клієнтами, проєктами та видами діяльності — це зручно для розподілу та включення до Ваших рахунків.

SAML

Увійдіть та зареєструйте користувачів із SAML через Вашого постачальника ідентифікації SSO на кшталт Google Workspace, Authentik або Azure Active Directory.

JSON API

Завдяки розширеному JSON API для читання та запису даних зовнішні програми та спеціальні інструменти можуть обмінюватися даними з Kimai.

Двофакторна аутентифікація (2FA)

Kimai підтримує двофакторну аутентифікацію з використанням TOTP-токенів: захистіть свої акаунти вже сьогодні!

Статистика

Ваша інформпанель покаже Вам деяку статистику щодо записаного часу та кількості зібраних даних. І ми завжди можемо додати інші графіки!

Docker

У нас є Docker-образ для розробки, а також декілька варіантів установки Docker-композиції для робочого використання.

Секундомір

Не хочете, щоб Ваші користувачі робили зайві записи в табелі? Перейдіть в режим секундоміра, який дозволяє лише "починати" або "зупиняти" запис часу.

Портал клієнта

Діліться статистикою клієнтів та проєктів, а також журналів робочого часу через секретне посилання, опційно захищене паролем.

Звіти та аналізи

Аналіз записаного часу за користувачами, клієнтами, проєктами, видами діяльності, мітками, періодами часу та іншим.
