Open Source

Kimai — це рішення для обліку часу з відкритим кодом. Реєструє робочий час і формує підсумок діяльності на вимогу. Щороку, щомісяця, щодня з групуванням за клієнтами, проєктами… Його перевага — це простота. Завдяки веб-інтерфейсу Kimai працює на різних платформах, навіть на мобільних пристроях.

З Kimai нудне заповнення електронних таблиць Excel робочим часом не тільки спрощується, але він також пропонує і десятки інших захоплюючих функцій, про які Ви навіть не думали, що їх не вистачає!

Підтримує ваш бізнес

Kimai має всі основні функції для налаштування вашого потоку обліку часу та інтеграції його з існуючою інфраструктурою.

Аутентифікація та безпека

Kimai підтримує зовнішні постачальники ідентичності та вхід через LDAP та SAML на кількох платформах (таких як Google Workspace, Azure AD або Authentik). Крім того, можливо включити двофакторну аутентифікацію за допомогою TOTP-токенів.

Звіти та аналізи

Аналіз записаних часів, клієнтів, проектів та видів діяльності. Оцінка за користувачами, клієнтами, проектами, видами діяльності, мітками, періодами часу та багато іншого.

Виставлення рахунків

Різноманітні шаблони рахунків-фактур (інвойсів), опції групування записів, налаштування формату номерів інвойсів та можливість додавати власні шаблони для експорту в форматах файлів: PDF, DOCX, HTML, XLSX, ODS.

JSON API

Завдяки розширеному JSON API для читання та запису даних зовнішні програми та Ваші власні інструменти можуть обмінюватися даними з Вашим улюбленим реєстратором часу.

Що Kimai може зробити для вас?

Kimai пропонує набагато більше, ніж просто облік часу, ось список інших важливих функцій для бізнесу.

Розширення

Kimai побудований з можливістю розширення функціональності — за допомогою плагінів-розширень. Розробники, які хочуть їх створити, можуть використовувати багато точок розширення.

Кілька користувачів

Kimai може використовуватися будь-якою кількістю користувачів: кожна інсталяція підтримує необмежену кількість користувачів з різними ролями (див. дозволи).

Мови

Kimai може бути локалізований на будь-яку мову і вже перекладений більш ніж на 30 мов.

Команди

Команди можуть бути створені для об'єднання користувачів у логічні групи на кшталт відділів або проєктних груп. До цих команд можуть бути додані клієнти та проєкти, що обмежить видимість і доступ для інших користувачів.

Клієнти

Керуйте своїми клієнтами та додавайте адресу, контакти, країну, валюту, видимість (для архівування) та багато інших полів.

Проєкти

Керуйте проєктами для кожного клієнта з номером замовлення, бюджетом та видимістю (для архівування).

Виставлення рахунків

Різноманітні шаблони рахунків-фактур (інвойсів), опції групування записів, налаштування формату номерів інвойсів та можливість додавати власні шаблони для експорту в форматах файлів: PDF, DOCX, HTML, XLSX, ODS.

Експортувати дані

Дані табеля можна експортувати в інші програми або надсилати у різні підрозділи у форматах XLSX (Excel), PDF, CSV та HTML. Експортовані записи, крім того, можна заблокувати та захистити від змін.

Видатки

Придбайте розширення для відстеження витрат за клієнтами, проєктами та видами діяльності — це зручно для розподілу та включення до Ваших рахунків.

SAML

Увійдіть і зареєструйте користувачів за допомогою SAML (Single sign on, також відомого як SSO) і повторно користуйтеся послугами постачальника ідентифікації такими як Google Workspace, Authentik або Azure Active Directory.

JSON API

Завдяки розширеному JSON API для читання та запису даних зовнішні програми та Ваші власні інструменти можуть обмінюватися даними з Вашим улюбленим реєстратором часу.

Двофакторна аутентифікація (2FA)

Kimai підтримує двофакторну аутентифікацію з використанням TOTP-токенів: захистіть свої облікові записи користувачів вже сьогодні!

Статистика

Ваша інформпанель покаже Вам деяку статистику щодо записаного часу та кількості зібраних даних. І ми завжди можемо додати інші графіки!

Docker

У нас є Docker-образ для розробки, а також декілька варіантів установки Docker-композиції для робочого використання.

Секундомір

Не хочете, щоб Ваші користувачі робили зайві записи в табелі? Перейдіть в режим секундоміра, який дозволяє лише "починати" або "зупиняти" запис часу. Більше не буде потрібно вводити вручну час початку та закінчення роботи.

Customer Portal

Share client and project statistics and timesheets with anyone using a secret URL, optionally password protected.
