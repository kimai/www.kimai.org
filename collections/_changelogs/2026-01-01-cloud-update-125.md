**Highlights**

- Expand `Monthly report` table: "only customer results" or "results including projects"
- Configurable rate rounding: `decimal`, `classic` (default) [see documentation]({{ site.url }}{% link _documentation/rounding.md %})
- Dark mode is now gray instead of dark blue with better contrast
- New `Automatic` option to switch between `Bright` and `Dark` theme

**Absence**

- New configuration to deactivate half-day vacation bookings
- Remove all fields except `comment` when editing an existing absence
- Editing a vacation comment failed, if all holidays were taken

**Invoice**

- Support for `Reverse Charge` and `Tax exempt` invoices
- Added `Customer` fields:
  - `Tax status` (Standard, Reverse charge, Exempt)
  - `Tax exemption reason` (free text, required in case of `Tax exempt`)
- Added `Peppol` profile (**beta**)
    - Currently supporting Vat-IDs in `cbc:EndpointID`
    - Limited to certain countries, please read [the documentation]({{ site.url }}{% link _documentation/plugin-invoice.md %}) 

**API**

- New API endpoint to lock months
- Retrieve and set break time via timesheet endpoints
- Added fields to Collection calls, to reduce the need of calling the entity endpoint
- Query `modified_after` in UTC
- Do not expose internal user preferences via API
