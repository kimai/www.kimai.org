**Important**

API passwords were deprecated two years ago and are now scheduled for removal.

Please read our blog post for details: [API passwords will be removed – switch to API tokens now]({% link _posts/en/2026-04-13-removing-api-passwords.md %})

**Improvements**

- Improved the monthly absence calendar (`Reports` > `Absence Calendar`)
    - Added support for public holidays
    - Added `view_team_absence` permission to allow teamleads to see absences of their team members
    - Highlight non-working days with a light gray background
- Disallow the characters `=` and `"` in user attributes and other name fields
- Avatar URLs must now be absolute URLs
- Do not copy rates when restarting a timesheet record
- Expenses: invoice queries using tags no longer return results
- Expenses: show exporter names in the dropdown
- SSO: ensure that RelayState redirects stay within the same cloud environment
- API: improved Team documentation
- API: prevent updates to disabled user preferences
- Importer: normalize tag names
- Kiosk: added configurable sound effects
- Kiosk: added HTML IDs for easier custom styling
- Updated translations
