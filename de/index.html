Authentifizierung und Sicherheit

Kimai unterstützt externe Identitätsanbieter und Anmeldung via LDAP und SAML über mehrere Provider (wie Google Workspace, Azure AD oder Authentik). Zwei-Faktor-Authentifizierung über TOTP-Tokens ist optional möglich.

Was kann Kimai für Sie tun?

Kimai bietet noch viel mehr als nur die Zeiterfassung, hier finden Sie eine Liste weiterer wichtiger Business-Funktionen.

Funktionen

Plugins

Kimai wurde mit Fokus auf Erweiterbarkeit entwickelt, so dass Funktionen mit Plugins hinzugefügt werden können. Entwickler, die eigene Plugins für Kimai erstellen, können auf eine umfangreiche API zugreifen.

Mehrere Benutzer

Kimai kann von beliebig vielen Benutzern verwendet werden: Jede Installation unterstützt eine unbegrenzte Anzahl von Benutzern mit unterschiedlichen Rollen (siehe Berechtigungen).

Mehrere Sprachen

Kimai kann in jede Sprache lokalisiert werden und ist bereits in mehr als 30 Sprachen übersetzt.

Teams

Teams teilen Ihre Benutzer in Gruppen ein. Schränken Sie den Zugriff auf Daten ein, indem Sie Ihren Teams Kunden und Projekte zuweisen.

Kunden

Verwalten Sie Ihre Kunden und legen Sie Adresse, Kontakte, Land, Währung, Sichtbarkeit (zur Archivierung) und viele weitere Felder fest.

Projekte

Verwalten Sie Projekte für jeden Kunden mit Auftragsnummer, Budget und Sichtbarkeit (zur Archivierung).

Rechnungen

Verschiedene Rechnungsvorlagen, Möglichkeiten zur Gruppierung von Einträgen, konfigurierbare Rechnungsnummern und verschiedene Formate wie PDF und DOCX.

Daten-Export

Stundenzettel können in verschiedenen Formaten exportiert werden. Exportierte Datensätze werden gesperrt und vor weiteren Änderungen geschützt.

Ausgaben

Behalten Sie den Überblick über Ihre Ausgaben. Diese Ausgaben können kategorisiert und in Ihre Rechnungen aufgenommen werden. Die Spesen- und Ausgabenverwaltung ist als kostenpflichtiges Plugin verfügbar.

SAML

Benutzer können sich mit SAML anmelden bzw. registrieren und dafür Ihren vorhandenen Identitätsanbieter wie Google Workspace, Authentik oder Azure Active Directory nutzen.

JSON-API

Mit einer umfangreichen JSON-API zum Lesen und Schreiben von Daten, so dass externe Anwendungen mit Ihrem bevorzugten Time-Tracker kommunizieren können.

Zwei-Faktor-Authentifizierung (2FA)

Kimai unterstützt die Zwei-Faktor-Authentifizierung mit TOTP-Tokens: sichern Sie Ihre Benutzerkonten noch heute ab!

Statistiken

Ihr Dashboard zeigt Ihnen mehrere Statistiken über aufgezeichnete Zeiten und die Menge der gesammelten Daten. Und wir können jederzeit weitere Diagramme hinzufügen!

Docker

Wir haben Docker-Images für die Entwicklung und es gibt mehrere Docker-Compose Varianten für den Produktionsbetrieb.

Stechuhr

Wollen Sie verhindern, das Ihre Benutzer manuelle Zeiteinträge erstellen? Wechseln Sie in den Stechuhr-Modus, der nur „Punch-In“- und „Punch-Out“-Aufnahmen zulässt.

Kundenportal

Geben Sie Kunden- und Projektstatistiken sowie Zeiterfassungsbögen über eine geheime, optional passwortgeschützte URL an andere weiter.

Berichte und Auswertungen

Analyse der erfassten Zeiten anhand von Benutzern, Kunden, Projekten, Aktivitäten, Tags, Zeiträumen und mehr.

E-Rechnungen

Erstellen Sie E-Rechnungen, sowohl als XML als auch PDF-A, in den Formaten EN16931, XRechnung 3.0, Zugferd/Factur-X und mit der Syntax CII & UBL
