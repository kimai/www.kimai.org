Kimai Store

The central place to find new plugins and commercial support for Kimai.

Any bundle, plugin and Kimai extension in here does only work with Kimai 0.9 and above, as the plugin mechanism was not existing before.

Due to this constraint and the fact, that 0.9 was at the time of this writing (2019-02-21) not even released, the amount of items in here is limited.

I hope to be able to feature more plugins in the near future.

Contribute

If you are a developer and want to create a new plugin, read the plugin documentation.

If you created a new piece of software (plugin, application ...) around Kimai 2 and want to see it listed here in the Store, read the store documentation.
