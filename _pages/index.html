Free time-tracking for everyone

Kimai is a free & open source timetracker. It tracks work time and prints out a summary of your activities on demand. Yearly, monthly, daily, by customer, by project … It’s simplicity is its strength. Due to Kimai’s browser based interface it runs cross-platform, even on your mobile device.

While there are many programs one can use for time and project tracking, they tend to be overkill for my rather simple needs. Kimai came about because I was tired to feed lame boring excel tables …

Kimai does not have to run while recording, you can even quit your browser. Your time is still recording until you stop it from any web browser that has access to your installation. It is designed to hold lots of users, but you can also use it for a single user if you want to.

Kimai can be localized to any language and we ship the following sets of translations: english, german, russian, italian, french, spanish, hungarian, arabic. Feel free to send your self-made language files – we’re looking for translators and would appreciate your support!

Feature list

The following list is not complete, but it should give you a quick overview of some features that Kimai v2 offers.

Name Description Free version
Multi-user Kimai v2 can be used by as many users as you want: each installation supports unlimited users with different roles (see permissions)
Mobile ready The layout is fully responsive, based on bootstrap and made for desktop and mobile usage. Every user can configure the data columns that should be shown.
Print invoices Different templates, calculator, invoice number generator and you can add your own templates, in the file formats: XLSX, ODS, CSV, DOCX
Manage customers Manage your customers and set address, contacts, country, currency, visibility (for archiving) and many more fields
Manage projects Manage projects for each customer with order number, budget and visibility (for archiving)
Manage activities Activities are used to record timesheet entries. Manage activities for each project with order number, budget and visibility (for archiving). Activities can be global, no more duplications necessary.
JSON API Featuring a JSON API so external apps can be created to communicate with your favorite time-tracker!
Docker support The release includes a docker image for development and we have a separate docker-compose setup for production
Archiving After a long usage a lot of entries can pile up in your dropdowns - now you can hide customers, projects and activities
Duration only If your local laws require you to only show durations instead of start and end date-time, you can activate that mode globally
User registration Kimai supports user self-registration with email activation and additionally passwort reset functionality
Weekend rates You can configure global settings for rate factors on single weekdays, e.g. 150% for saturday and sunday
Statistics Your dashboard will show you several statistics about recorded times and the amount of collected data. We can add more charts, just ask us!
Export timesheets User can be export their timesheets for keeping offline record of their worktime
Rate calculation Rates can be calculated based on hourly and fixed amounts, configurable for customers, projects, activities and each timesheet record individually.
Multiple records You can record as many times as you want in parallel, the limit can be configured
Permissions Assign different roles to your user: User can record times, teamleads can print invoices and configure project/activities, admins can configure users and customers (read more)
Filter times You can filter all times by user, project, activity and date
Different themes Kimai bundles different color themes and display settings, so every user can tweak it to his personal flavour
User profile Personal statistics and per user settings for hourly rate, Kimai theme, password and avatar
Migrate from v1 Data from Kimai v1 can be imported, for a seamless migration of your time-tracker (read more)
Extendable Kimai v2 was built with extendability in mind, it is extendable with Symfony bundles and has many extension points
Modern technologies Based on newest technologies and frameworks: PHP7, Symfony 4, Doctrine, Flex, Bootstrap, Webpack and many more...
SQLite support No database setup required, Kimai has SQLite support out-of-the-box. And we support MySQL/MariaDB as well.
