技術支援

一些使用者偏好在內部環境執行 Kimai，但又不想錯過專業的支援。我很願意協助您並為自架的安裝實例提供支援方案。

直接聯繫

提供電子郵件支援與視訊通話予付費使用者。

我提供具競爭力的收費標準，來讓您專注於您的業務。我將協助您解決問題以及安裝與升級您的 Kimai 安裝實例。

雲端服務使用者

提供電子郵件支援予雲端使用者。

雲端使用者一般可以在 24 小時內，在中歐時間（CET）週一至週五的 10:00 至 16:00 之間（不包括法定假日）收到回覆。

建立詢問單

社群

未提供電子郵件支援予免費的自架安裝實例。

GitHub 上的支援社群通常會迅速回應。請先從既有的 issues 中搜尋 ，看看您的問題是否已得到解決。

造訪 GitHub

透通畫面分享視訊通話提供支援

如果您需要 Kimai 的介紹或更新等支援，您可以預約與開發者 Kevin Papst 進行有償的畫面分享視訊通話。 選擇時段後，您將被重新導向到 Stripe 付款頁面，之後您將收到通話的邀請。

如果您屆時將無法參與通話，您必須提前至少 24 個小時取消預約以便重新安排時程。不提供退款。
