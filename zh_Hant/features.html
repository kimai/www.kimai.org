透過外掛程式取得更多功能

技術與業務支援

我確保對所有我所建構的應用程式提供支援，我將協助您與您的團隊。

一起合作

Kimai 能為您做什麼呢？

Kimai 不僅提供工時追蹤，還有其它重要的業務功能。

行動裝置友善

版面配置完全採用回應式設計，同時適用於桌上型電腦與行動裝置。每位使用者可以設定要顯示的資料縱行，可避免在小型裝置上需要橫向捲動等情況。

認證與安全

Kimai 支援外部身份識別提供者，可從多個提供者（例如 Google Workspace、Azure AD、Authentik）透過 LDAP 與 SAML 進行登入。此外還可透過 TOTP token 啟用雙重要素身份認證。

外掛程式

Kimai 具有可擴充的設計，可透過外掛程式增加功能。想要建立外掛程式的開發人員能夠利用多個擴充點。

多使用者

Kimai 允許任意數量的使用者：每個已安裝實例都支援無上限的使用者與各種團隊角色（請參閱「權限」）。

多語言

Kimai 可以在地化為任何語言，目前已獲翻譯為至少 30 種語言。

團隊

可以建立團隊以將您的使用者合理分組，例如依部門或與專案相關的同儕小組分組。可將客戶與專案指派給這些團隊，以限制對其他使用者的可見度與存取權。

客戶

管理您的客戶並設定地址、聯絡人、國家、貨幣、可見度（用於封存）、與更多其它欄位。

專案

透過訂單編號、預算、與可見度（封存用）來管理每位客戶的專案。

活動

活動用於將工作單元指派到工時單中的項目。透過訂單編號、預算、與可見度（封存用）來管理每項專案的活動。活動可以設定為全域，因此多項專案共用同一個工作單元時無須複製活動。

收支單

各種收支單模版、項目分組選項、可設定的收支單編號，並可新增自己的 PDF、DOCX、HTML、XLSX、與 ODS 檔案格式的模版。

匯出資料

工時單資料可以匯出給其它程式處理，或是使用 XLSX (Excel)、PDF、CSV、與 HTML 檔案格式與不同部門共用。還可以鎖定已匯出的紀錄以防止進一步更改。

開銷管理

購買該外掛程式後，可以根據客戶、項目、與活動來追蹤開銷 ，以便分類並列入您的收支單。

SAML

利用 SAML 式單一登入（SSO），透過您現有的身份識別提供者，例如 Google Workspace、Authentik、Microsoft Entra ID，來登入與註冊。

JSON API

具有能用以讀取資料的全面 JSON API，因此外部應用程式與您的自訂工具能與您所喜好的工時追蹤工具溝通。

雙重要素驗證（2FA）

Kimai 支援使用 TOTP token 進行雙重要素身份認證（2FA）：立即保護您的使用者帳號！

統計資訊

您的儀錶板會顯示有關已記錄工時與收集資料量等的統計資訊。而我們能持續新增更多種圖表的支援！

Docker

我們有用於開發的 Docker 映像，還有為正式用途所設定的多種 Docker-Compose 變體。

打卡鐘模式

不希望使用者任意登記工時表紀錄嗎？可改用打卡鐘模式，該模式只允許「打卡上班」與「打卡下班」的紀錄。使用者不再需要手動指定開始與結束時間。

費率計算

費率可按工時或按次計算，可針對客戶、專案、活動、與各工時紀錄各別設定。

LDAP

將 Kimai 與您的公司目錄伺服器（LDAP 或 Active Directory）連接並使用您現有的身份驗證基礎架構。

版本遷移

支援匯入 Kimai v1 的資料，以便從您的工時追蹤工具無縫遷移。

進行中的紀錄數

可允許平行記錄任意次，也可限制為一次。

上午／下午格式

Kimai 使用在地化的 24 小時制格式。但若您較習慣上午／下午格式，您可調整相關設定。

僅列出持續時長

如果您當地的法律要求您只列出持續時長而不是列出開始與結束時間，您可以全域啟用本模式。

工時表篩選器

您可依使用者、客戶、專案、活動、與日期篩選所有的工時紀錄。

權限

可自訂的權限架構。指派不同的角色予您的使用者，例如一般使用者能記錄工時，組長能列印收支單與設定專案與活動，管理員能設定使用者與客戶。

使用者資料

個人統計資料與個別使用者的每小時費率、Kimai 顯示主題、密碼、與個人資料相片等設定。

使用者註冊

Kimai 支援使用者透過以電子郵件啟用帳號的流程來自行註冊。

密碼重設

忘記密碼了嗎？Kimai 可協助您透過電子郵件來重設密碼。

Customer Portal

Share client and project statistics and timesheets with anyone using a secret URL, optionally password protected.

Dark mode

Because every app needs a dark-mode, which helps to reduce eye strain in the evening or in dark environments.

報告與分析

分析已記錄的工時、客戶、專案、活動。可根據使用者、客戶、專案、活動、標籤、時間區間、等條件進行評級。

Knows no limits

Your SaaS software always includes limits? Not Kimai: there are no limits. You only pay for the amount of active users. I don't limit you, and I do not charge for inactive users. You can create an unlimited amount of users, customers, projects, activities, timesheets and more.

Privacy comes first

I respect your and your users privacy and data. No, this is not yet another boring introduction into a Privacy policy - that's how I run this business. I don't sell data. I follow the GDPR and will answer any question about involved sub-processors.

Security in Kimai

Kimai implements best-practices in application design to prevent common attacks. Automated security controls are critical to any software development workflow to reduce the amount of time from defect creation to defect detection. These scans run on every code change, every time.

Security in the platform

Software is only as secure as the system it runs on. I take a defense-in-depth approach with server and network infrastructure. Customers on the hosted platform are secured through multiple layers of protection.
