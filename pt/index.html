Um software, duas versões

O Kimai é um software de código aberto disponível para auto-hospedagem e também como SaaS.

Open Source

O Kimai é uma solução de código aberto para controle de tempo. Rastreia o tempo de trabalho e imprime um resumo das suas atividades sob demanda. Anualmente, mensalmente, diariamente, por cliente, por projeto… A sua simplicidade é a sua força. Pelo fato da interface do Kimai ser à base de qualquer navegador da internet, funciona em várias plataformas, mesmo no seu aparelho móvel.

Com o Kimai, o tedioso processo de alimentar planilhas do Excel usando as suas horas de trabalho não é apenas simplificado, mas também oferece dezenas de outros recursos interessantes que nem sabe que perde até agora!

Suporta seu negócio

O Kimai possui todos os recursos essenciais de negócios de que você precisa para configurar seus fluxos de trabalho de rastreamento de tempo e integrá-los à sua infraestrutura existente.

Autenticação e Segurança

O Kimai oferece suporte a provedores de identidade externos e login via LDAP e SAML em vários provedores (como Google Workspace, Azure AD ou Authentik). Além disso, a autenticação de dois fatores pode ser habilitada por meio de tokens TOTP.

Relatórios e análises

Análise de tempos registrados, clientes, projetos e atividades. Avaliação de acordo com usuários, clientes, projetos, atividades, tags, períodos de tempo e muito mais.

Faturamento

Diferentes modelos de fatura, opções para agrupamento das entradas, fatura com números configuráveis e pode adicionar os seus próprios modelos em formato PDF, DOCX, HTML, XLSX e ODS.

API JSON

Apresentando uma extensa API JSON para leitura e escrita de dados, para que aplicativos externos e suas ferramentas personalizadas possam se comunicar com seu rastreador de tempo favorito.

O que o Kimai pode fazer por você?

O Kimai oferece muito mais do que apenas rastreamento de tempo, aqui está uma lista de outras funções empresariais importantes.

Plug-ins

O Kimai foi desenvolvido para ser extensível e os recursos podem ser adicionados através de plug-ins. Os programadores que desejam criá-los podem usar muitos pontos de extensão.

Multiutilizador

O Kimai pode ser usado por quantos utilizadores quiser: cada instalação não possui um limite de utilizadores e com diferentes funções (consulte permissões).

Vários idiomas

O Kimai pode ser localizado em qualquer idioma e já está traduzido para mais que 30 línguas.

Equipas

As equipas podem ser criadas para agrupar os seus utilizadores em grupos lógicos, como departamentos ou grupos de pares relacionados ao projeto. Os clientes e os projetos podem ser atribuídos a estas equipas, o que limitará a visibilidade e o acesso dos outros utilizadores.

Clientes

Gira os seus clientes e defina um endereço, um país, uma moeda, visibilidade (para arquivamento), contatos e muitos outros campos.

Projetos

Gira os projetos para cada cliente com número de pedido, orçamento e visibilidade (para arquivamento).

Faturamento

Diferentes modelos de fatura, opções para agrupamento das entradas, fatura com números configuráveis e pode adicionar os seus próprios modelos em formato PDF, DOCX, HTML, XLSX e ODS.

Exportar dados

Os dados do quadro de horários podem ser exportados para outros programas ou podem ser compartilhados entre diferentes departamentos nos formatos XLSX (Excel), PDF, CSV e HTML. Os registos exportados também podem ser bloqueados e protegidos contra outras alterações.

Despesas

Compre o plug-in e acompanhe as despesas com base no cliente, no projeto e na atividade - para a categorização e para a inclusão nas suas faturas.

SAML

Faça login e registe os utilizadores via SAML (single sign on, também conhecido como SSO) e reutilize o seu provedor de identidade já existente, como o Google workspace, Authentik ou o active directory Azure.

API JSON

Apresentando uma extensa API JSON para leitura e escrita de dados, para que aplicativos externos e suas ferramentas personalizadas possam se comunicar com seu rastreador de tempo favorito.

Autenticação de Dois Fatores (2FA)

O Kimai oferece suporte à autenticação de dois fatores com tokens TOTP: proteja suas contas de usuário hoje!

Estatísticas

O seu painel mostrará várias estatísticas sobre os tempos registados e a quantidade dos dados coletados. E sempre poderemos adicionar mais gráficos!

Docker

Temos imagens do Docker para desenvolvimento e diferentes variações de configurações do Docker-Compose para uso em produção.

Relógio de ponto

Não deseja que os seus utilizadores registem um quadro de horários arbitrários? Mude para o modo relógio-de-ponto, isso permite apenas os registos “de entrada” e “de saída”. Chega de horários de início e término fornecidos manualmente.
