Что может сделать Kimai для вас?

Kimai предлагает гораздо больше, чем просто учет времени, вот список других важных бизнес-функций.

Плагины

Kimai построен с возможностью расширения набора функций с помощью плагинов. Разработчик, который хочет создать плагин для Kimai, может использовать множество точек расширения.

Многопользовательский

Kimai может использоваться любым количеством пользователей: каждая установка поддерживает неограниченное количество пользователей с различными ролями (см. разрешения).

Многоязычность

Kimai может быть переведён на любой язык, и уже переведён более чем на 30 языков.

Команды

Команды могут быть созданы для объединения пользователей в логические группы, такие как отделы или группы коллег по проекту. К этим командам могут быть прикреплены клиенты и проекты, что ограничит видимость и доступ для других пользователей.

Клиенты

Управляйте своими клиентами и добавляйте адрес, контакты, страну, валюту, видимость (для архивирования) и многие другие поля.

Проекты

Управляйте проектами для каждого клиента с номером заказа, бюджетом и видимостью (для архивирования).

Выставление счетов

Различные шаблоны, калькулятор, генератор номеров счетов, а также можете добавить свои собственные шаблоны для экспорта в форматы файлов: PDF, DOCX, HTML, XLSX, ODS.

Экспорт данных

Данные табеля учёта рабочего времени можно экспортировать для использования в других программах или для передачи в различные отделы в форматах: XLSX (Excel), PDF, CSV и HTML. Кроме того, экспортированные записи могут быть заблокированы и защищены от дальнейших изменений.

Расходы

Отслеживайте свои расходы по клиентам, проектам и активностям. Эти расходы можно распределять по категориям и включать в счета-фактуры. Обратите внимание: отслеживание расходов доступно только в платном плагине.

SAML

Войдите в систему и зарегистрируйте пользователей с помощью SAML (Single sign on, известный также как SSO) и пользуйтесь существующим поставщиком идентификации, например, Google Workspace, Authentik или Azure Active Directory.

JSON API

Благодаря наличию JSON API для чтения и записи данных внешние приложения и Ваши собственные инструменты могут взаимодействовать с Вашим любимым регистратором времени.

Двухфакторная аутентификация (2FA)

Kimai поддерживает двухфакторную аутентификацию с использованием TOTP-токенов: обеспечьте безопасность ваших учетных записей сегодня!

Статистика

Ваша информпанель покажет вам некоторую статистику о записанном времени и о количестве собранных данных. И мы всегда можем добавить другие диаграммы!

Docker

Выпуск включает в себя образ Docker для разработки и отдельную установку Docker-compose для рабочей среды.

Секундомер

Не хотите, чтобы ваши пользователи регистрировали произвольные записи в табеле учёта рабочего времени? Переключитесь в режим секундомера, который позволяет делать только записи «старт» и «стоп». Больше не нужно будет вручную указывать время начала и окончания работы.
