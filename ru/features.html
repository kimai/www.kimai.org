Что может сделать Kimai для вас?

Kimai предлагает гораздо больше, чем просто учет времени, вот список других важных бизнес-функций.

Поддержка мобильных устройств

Макет полностью адаптивный, созданный для настольных и мобильных устройств. Каждый пользователь может настроить столбцы данных, которые должны отображаться, например, для избежания горизонтальной прокрутки на небольших устройствах.

Аутентификация и безопасность

Kimai поддерживает внешние поставщики идентификации и вход через LDAP и SAML на нескольких платформах (например, Google Workspace, Azure AD или Authentik). Кроме того, можно включить двухфакторную аутентификацию с использованием TOTP-токенов.

Плагины

Kimai построен с возможностью расширения набора функций с помощью плагинов. Разработчик, который хочет создать плагин для Kimai, может использовать множество точек расширения.

Многопользовательский

Kimai может использоваться любым количеством пользователей: каждая установка поддерживает неограниченное количество пользователей с различными ролями (см. разрешения).

Многоязычность

Kimai может быть переведён на любой язык, и уже переведён более чем на 30 языков.

Команды

Команды могут быть созданы для объединения пользователей в логические группы, такие как отделы или группы коллег по проекту. К этим командам могут быть прикреплены клиенты и проекты, что ограничит видимость и доступ для других пользователей.

Клиенты

Управляйте своими клиентами и добавляйте адрес, контакты, страну, валюту, видимость (для архивирования) и многие другие поля.

Проекты

Управляйте проектами для каждого клиента с номером заказа, бюджетом и видимостью (для архивирования).

Активности

Активности используются для присвоения рабочих ресурсов записям в табеле. Управляйте активностями в каждом проекте с указанием номера заказа, бюджета и видимости (для архивирования). Активности могут быть глобальными, что исключает необходимость дублирования, если несколько проектов используют один и тот же рабочий ресурс.

Выставление счетов

Различные шаблоны, калькулятор, генератор номеров счетов, а также можете добавить свои собственные шаблоны для экспорта в форматы файлов: PDF, DOCX, HTML, XLSX, ODS.

Экспорт данных

Данные табеля учёта рабочего времени можно экспортировать для использования в других программах или для передачи в различные отделы в форматах: XLSX (Excel), PDF, CSV и HTML. Кроме того, экспортированные записи могут быть заблокированы и защищены от дальнейших изменений.

Расходы

Отслеживайте свои расходы по клиентам, проектам и активностям. Эти расходы можно распределять по категориям и включать в счета-фактуры. Обратите внимание: отслеживание расходов доступно только в платном плагине.

SAML

Войдите в систему и зарегистрируйте пользователей с помощью SAML (Single sign on, известный также как SSO) и пользуйтесь существующим поставщиком идентификации, например, Google Workspace, Authentik или Azure Active Directory.

JSON API

Благодаря наличию JSON API для чтения и записи данных внешние приложения и Ваши собственные инструменты могут взаимодействовать с Вашим любимым регистратором времени.

Двухфакторная аутентификация (2FA)

Kimai поддерживает двухфакторную аутентификацию с использованием TOTP-токенов: обеспечьте безопасность ваших учетных записей сегодня!

Статистика

Ваша информпанель покажет вам некоторую статистику о записанном времени и о количестве собранных данных. И мы всегда можем добавить другие диаграммы!

Docker

Выпуск включает в себя образ Docker для разработки и отдельную установку Docker-compose для рабочей среды.

Секундомер

Не хотите, чтобы ваши пользователи регистрировали произвольные записи в табеле учёта рабочего времени? Переключитесь в режим секундомера, который позволяет делать только записи «старт» и «стоп». Больше не нужно будет вручную указывать время начала и окончания работы.

Расчёт тарифа

Ставки могут рассчитываться на основе почасовых и фиксированных сумм, настраиваться для клиентов, проектов, видов активности и для каждой записи табеля в отдельности.

LDAP

Подключите Kimai к серверу каталогов своей компании (LDAP или Active Directory) и используйте существующую инфраструктуру аутентификации.

Миграция

Данные из Kimai вер. 1 могут быть импортированы для беспрепятственной миграции вашего регистратора времени.

Количество активных записей

Вы можете вести параллельную запись сколько угодно раз, но при этом можно настроить допустимый лимит активных записей до 1.

Формат AM/PM

Kimai использует локализованную версию 24-часового формата. Но если вы привыкли к формату am/pm, можете настроить его.

Только продолжительность

Если местные законы требуют, чтобы вы показывали только длительность вместо времени начала и окончания, вы можете активировать этот режим глобально.

Фильтры табеля

Вы можете фильтровать все записи времени по пользователю, клиенту, проекту, активности и дате.

Разрешения

Гибкая структура прав доступа. Назначайте пользователям различные роли, например, пользователи могут регистрировать время, лидеры команд могут печатать счета и настраивать проекты/активности, администраторы могут настраивать учётные записи пользователей и клиентов.

Профиль пользователя

Личная статистика и настройки почасовой оплаты, темы Kimai, пароля и аватара для каждого пользователя.

Регистрация пользователя

Kimai поддерживает самостоятельную регистрацию пользователей с активацией по электронной почте.

Сброс пароля

Забыли пароль? Нет проблем, Kimai позаботился о вас, предоставив возможность сброса пароля по электронной почте.

Customer Portal

Share client and project statistics and timesheets with anyone using a secret URL, optionally password protected.

Dark mode

Because every app needs a dark-mode, which helps to reduce eye strain in the evening or in dark environments.

Отчеты и анализы

Анализ записанных временных данных, клиентов, проектов и активностей. Оценка по пользователям, клиентам, проектам, активностям, меткам, временным периодам и многому другому.

Knows no limits

Your SaaS software always includes limits? Not Kimai: there are no limits. You only pay for the amount of active users. I don't limit you, and I do not charge for inactive users. You can create an unlimited amount of users, customers, projects, activities, timesheets and more.

Privacy comes first

I respect your and your users privacy and data. No, this is not yet another boring introduction into a Privacy policy - that's how I run this business. I don't sell data. I follow the GDPR and will answer any question about involved sub-processors.

Security in Kimai

Kimai implements best-practices in application design to prevent common attacks. Automated security controls are critical to any software development workflow to reduce the amount of time from defect creation to defect detection. These scans run on every code change, every time.

Security in the platform

Software is only as secure as the system it runs on. I take a defense-in-depth approach with server and network infrastructure. Customers on the hosted platform are secured through multiple layers of protection.
